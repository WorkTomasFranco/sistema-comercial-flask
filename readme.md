# Sistema de Gestión Comercial - Proyecto Portfolio

Aplicación web desarrollada con Flask para la gestión de productos, ventas y análisis comercial.

## Funcionalidades

- Gestión de productos (alta y visualización)
- Control de stock con alerta de stock mínimo
- Registro de ventas
- Cálculo automático de ganancia
- Dashboard con métricas comerciales
- Gráfico de ventas por producto

## Tecnologías utilizadas

- Python
- Flask
- SQLite
- Bootstrap
- Chart.js

## Estructura del sistema

- Productos
- Ventas
- Detalle de ventas
- Reportes comerciales

## Objetivo del proyecto

Simular un sistema real de gestión para pequeños comercios, aplicando conceptos de:

- Modelado relacional
- Lógica de negocio
- Validaciones
- Métricas comerciales
- Arquitectura web básica
